# Caffy - POS & Inventory Management System ☕

Caffy is a comprehensive mobile application designed to streamline point-of-sale (POS) operations and real-time inventory management for small-scale coffee shops.

## 🚀 Key Features
* **Real-time Inventory Tracking:** Automatic stock deduction upon successful orders.
* **POS System:** Intuitive interface for staff to process payments and manage orders.
* **Cross-Platform Support:** Fully functional and tested on both iPhone and iPad layouts.
* **Backend Integration:** Secure data storage and authentication using Firebase.

## 🧪 Quality Assurance & UX Research Focus
As a QA-focused developer, I prioritized software quality and user experience throughout the SDLC:
* **Usability Testing:** Conducted research with 8 users (aged 29-55) to evaluate the "Copayment" and "Stock Management" features.
* **UX Metrics:** Achieved a high satisfaction level based on **System Usability Scale (SUS)** score analysis.
* **Test Documentation:** Developed comprehensive **Test Scenarios** and **Test Cases** to validate critical business logic.
* **Information Architecture:** Designed **Sitemaps** and **User Flows** to ensure a seamless navigation experience.

## 🛠 Tech Stack
* **Framework:** Flutter
* **Language:** Dart
* **Database:** Firebase Firestore
* **Design & Analysis:** Figma, FigJam, Power BI
